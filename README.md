# WorshipStudio Asaf 148

**WorshipStudio Asaf 148** es una plataforma integral de gestión y planificación para ministerios de alabanza. Diseñada para reemplazar los flujos de trabajo basados en Excel, permite una coordinación fluida entre Directores, Cantores y Músicos.

## ✨ Características Principales

- **Matriz de Planificación (Matrix View)**: Vista panorámica mensual para la asignación rápida de roles y balance de carga de trabajo.
- **Flujo de Aprobación Refinado**: Sistema de estados (Borrador, Revisión, Aprobado, Rechazado) para garantizar la calidad del repertorio.
- **Repertorio Inteligente**: Base de datos centralizada con detección de duplicados mensuales, gestión de tonos vocales y enlaces de ensayo.
- **Exportación Profesional**: Generación de reportes en PDF (estilo tarjetas) y Excel para distribución y archivo.
- **Diseño Premium**: Interfaz moderna con modo oscuro, glassmorphism y efectos neon para una experiencia de usuario superior.

## 🚀 Inicio Rápido

1. **Instalar dependencias**:
   ```bash
   npm install
   ```
2. **Configurar Firebase**: Asegúrate de tener las credenciales de Firestore en tu entorno.
3. **Ejecutar servidor de desarrollo**:
   ```bash
   npm run dev
   ```
   Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
